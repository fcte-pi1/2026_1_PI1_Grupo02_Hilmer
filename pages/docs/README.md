# Grupo 02 (Hilmer): Pikachu

## **Introdução**

Esse repositório abriga a documentação das atividades desenvolvidas pelo **Grupo 02** na disciplina de *Projeto Integrador de Engenharia 1*, com o professor **Hilmer Rodrigues**, na *UnB - FCTE*. De forma resumida, foi orientado que o grupo seguisse uma abordagem *Scrum*, com **sprints semanais**, para o desenvolvimento de um **micromouse** capaz de resolver labirintos simples, ao longo do semestre *2026/1*. Para isso, o grupo se subdividiu em [núcleos](#membros-do-grupo-02).
